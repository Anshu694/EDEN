# eden-apps
 Firmware for EDEN modules
 
### WiFi Config
 ```
 mos call wifiSettings '{"ssid": "eden", "pass": "eden1234"}'
 ```


#### AP name changed for different modules
* 4 Switch          - EDEN_4
* 3 Switch          - EDEN_3
* 2 Switch          - EDEN_2
* Power Switch      - EDEN_P
* Fan Module        - EDEN_4F
* RGB Module        - EDEN_RGB
* Curtain           - EDEN_CURTAIN
* Dimmer            - EDEN_4
* Door Sensor       - EDEN_DS
